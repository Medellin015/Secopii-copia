# Secopii-Procesos

Buscador de **licitaciones y procesos de contratación** del SECOP II sobre los datos abiertos de Colombia (datos.gov.co, dataset `p6dx-8zbt`).

- Al abrir, muestra automáticamente las licitaciones publicadas en el **último mes**, de la más reciente a la más antigua.
- La **contratación directa está excluida** de todas las consultas: vista inicial, búsquedas con filtros y descargas a Excel.
- Aplicación estática (HTML/CSS/JS sin dependencias de compilación); consulta la API SODA directamente desde el navegador.
- Instructivo de uso en [`instructivo.pdf`](instructivo.pdf).
